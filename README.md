🌤️ Interactive Day-Night Weather Widget
A clean, minimalist, and interactive web component that allows users to toggle between Day and Night modes with smooth animations.

🚀 Features
Dynamic Transitions: Smooth color switching between a bright, clean Skyblue and a premium Deep Navy background.

Interactive UI: Includes hover glow effects on the Sun/Moon and pulse feedback on click.

Fluid Animations: Uses GSAP (GreenSock Animation Platform) for floating clouds, entrance effects, and star generation.

Responsive Design: Optimized for mobile and desktop screens using CSS Media Queries.

🛠️ Tech Stack
HTML5 & CSS3: Structured layout with CSS Variables for easy theme management.

JavaScript (ES6+): Handles logic, star generation, and event listeners.

GSAP: Powering all high-performance animations and transitions.

💻 How to Run
Clone this repository to your local machine.

Open index.html in any modern web browser.

Click on the scene to toggle between Day and Night!
